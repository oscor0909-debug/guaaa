# GuaTravel 旅遊手帳

這是一個溫馨的動物森友會風格團體旅遊規劃 App。

## 功能特色

*   **行程規劃**：視覺化的時間軸行程表。
*   **預訂管理**：集中管理機票、住宿、交通與票券憑證。
*   **記帳分帳**：支援多幣別（TWD/JPY）記帳，自動計算分攤金額與建議轉帳。
*   **行前準備**：檢查行李與待辦事項。
*   **團體協作**：邀請旅伴加入，共同編輯。

## 技術堆疊

*   React 18
*   Vite
*   Tailwind CSS
*   Firebase (Firestore, Auth, Storage)
*   PWA (Progressive Web App)

## 部署

本專案設定為可部署於 Cloudflare Pages。
